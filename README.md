# Еда со скидкой

Fullstack приложение для покупки еды из ресторанов со скидкой в конце рабочего дня.

## Описание

Приложение позволяет ресторанам продавать остатки еды в конце дня со скидкой, чтобы не выбрасывать продукты. Пользователи могут просматривать рестораны, выбирать блюда и оформлять заказы.

## Технологии

### Frontend:
- React 18
- SCSS
- React Router (планируется)

### Backend:
- Node.js + Express (планируется)
- MongoDB / PostgreSQL (планируется)

## Установка и запуск

### Установка зависимостей:
```bash
npm install
```

### Запуск в режиме разработки:
```bash
npm start
```

Приложение откроется на [http://localhost:3000](http://localhost:3000)

### Сборка для продакшена:
```bash
npm run build
```

## Структура проекта

```
src/
```

## Функционал (в разработке)

### Для пользователей:
- Просмотр списка ресторанов
- Фильтрация ресторанов (планируется)
- Просмотр меню ресторана (планируется)
- Добавление в корзину (планируется)
- Оформление заказа (планируется)
- История заказов (планируется)
- Регистрация и авторизация (планируется)

### Для ресторанов (админка):
- Панель управления (планируется)
- Управление меню (планируется)
- Просмотр заказов (планируется)
- Статистика (планируется)
